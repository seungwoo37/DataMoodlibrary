datamood.audio package
======================

Datamood의 오디오 처리 모듈로, 음성 신호 분석·전처리부터 
음성 기반 감정(무드) 분류까지 지원합니다.  
오디오 데이터를 활용한 AI 감정 분석 워크플로우 구축에 필요한 핵심 기능들이 포함되어 있습니다.


audio_mood Module
---------------------------------

오디오 데이터를 입력받아 감정 상태를 분류하는 핵심 분석 모듈입니다.
특징 추출, 스펙트럼 기반 분석, 모델 기반 감정 분류 기능 등을 제공합니다.
